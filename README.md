# M-Motors - Frontend

Interface React pour la plateforme de vente et location de véhicules M-Motors.

## Stack technique

- React
- Vite
- Bootstrap

## Installation

```bash
git clone https://github.com/ANA1512/m-motors-react-frontend.git
cd m-motors-react-frontend
npm install
npm run dev
```

L'application tourne sur `http://localhost:5173`

## Fonctionnalités

- Affichage de la liste des véhicules
- Recherche par nom en temps réel

## Prérequis

Le backend doit tourner sur `http://localhost:5001`
